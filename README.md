# 呢喃课表导入 for Android and iOS

一个基于 Flutter 开发的课表导入工具。

本项目通过官方统一认证网页完成登录，自动拉取当前学期课表，并将课程与考试事件写入手机系统日历，方便在原生日历中统一查看和管理。

基于 ArkTS 开发的 HarmonyOS 应用https://github.com/121mc/NJU_Calendar_Importer_ArkTS

Android 下载链接https://box.nju.edu.cn/d/adf190c64ece4373bf43/

iOS 下载方式：在 APP Store 搜索 呢喃课表导入

## 功能特性

- 支持本科生 / 研究生登录
- 使用官方网页登录统一认证
- 自动拉取当前学期课表
- 本科生支持同时导入期末考试
- 支持选择目标系统日历并一键写入
- 支持覆盖删除本应用此前导入的旧事件
- 支持一键清空本应用导入的全部日历事件


## 使用方法

1. 打开应用并完成统一认证登录。
2. 拉取当前学期课表。
3. 加载手机系统日历并选择目标日历。
4. 点击“写入系统日历”完成导入。
5. 如需清理，可点击“一键清空本应用导入事件”。


## 权限说明

本应用需要以下权限：

- **网络权限**：用于访问统一认证和课表接口。
- **日历权限**：用于读取系统日历列表、写入课表事件，以及删除本应用此前导入的旧事件。
